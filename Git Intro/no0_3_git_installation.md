# 如何取得 Git 0.3: 下載與安裝

要開始使用 Git，您需要先在您的電腦上安裝它。

---

## 官方網站

取得 Git 最可靠的來源永遠是官方網站：

> **[https://git-scm.com/](https://git-scm.com/)**

網站會自動偵測您的作業系統，包含 Windows, macOS, 或 Linux，並提供對應的下載連結。

值得注意的是，Git 的官方網站 `git-scm.com` 是一個由 Git 社群維護的資源，而非由單一企業，例如 Microsoft，直接擁有或營運。儘管 Microsoft 是 Git 生態系的重要貢獻者，但官方網站的獨立性確保了其作為中立資訊來源的地位。

## 您下載的是什麼？

當您從官網下載並安裝 Git 時，您主要得到的是 **Git 核心命令列工具 Core Command-Line Tool**。

這是一個可以在終端機 Terminal 或 Command Prompt 中執行的程式。所有 Git 的功能，無論多麼複雜，都源自於這個核心工具。這也是我們接下來所有筆記將會圍繞的核心。
