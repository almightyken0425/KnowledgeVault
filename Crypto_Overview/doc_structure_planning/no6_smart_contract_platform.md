# 第二階段：智能合約平台

## 階段目標

探討以太坊作為智能合約平台的核心技術，理解如何在區塊鏈上執行可程式化的去中心化應用。

---

## 核心主題

### 智能合約基本概念

- 智能合約定義與運作原理
- 自動執行與不可篡改特性
- 與比特幣腳本語言的差異
- 智能合約的應用場景與限制

### 以太坊架構

- 以太坊設計理念與創新
- 帳戶模型 vs UTXO 模型
- World State 全局狀態概念
- 狀態轉換函數
- 以太坊虛擬機 EVM 簡介

### EVM 虛擬機制

- EVM 架構與執行環境
- Opcodes 操作碼系統
- Stack、Memory、Storage 記憶體模型
- 執行上下文與呼叫堆疊
- 圖靈完備性與停機問題

### Gas 機制

- Gas 的設計目的
- Gas Price、Gas Limit、Gas Used 關係
- 交易費用計算公式
- EIP-1559 費用改革機制
- Base Fee、Priority Fee、Max Fee
- MEV 礦工可提取價值問題

### Solidity 語言基礎

- Solidity 語言特性
- 合約結構與生命週期
- 資料型別與儲存位置
- 函數可見性與修飾器
- 事件與日誌機制
- 繼承與介面
- 安全性考量與常見漏洞
