# 第十階段：NFT 非同質化代幣

## 階段目標

探討 NFT 技術原理與應用，理解如何在區塊鏈上表示獨一無二的數位資產。

---

## 核心主題

### NFT 基礎概念

- Non-Fungible Token 定義
- 非同質化 vs 同質化代幣差異
- 唯一性與稀缺性特性
- 所有權證明與可驗證性
- 應用場景分析
  - 數位藝術與收藏品
  - 遊戲道具與虛擬資產
  - 域名與身份
  - 票務與會員資格
  - 實體資產代幣化
- NFT 市場規模與發展

### NFT 標準

- ERC-721 標準詳解
  - Token ID 唯一性
  - 所有權轉移函數
  - 授權與批准機制
  - Metadata 元數據結構
- ERC-1155 多代幣標準
  - 同質化與非同質化混合
  - 批量轉移效率優化
  - 遊戲資產應用場景
- ERC-721 vs ERC-1155 比較
- Token URI 與元數據指向
- OpenSea Metadata 標準

### NFT 儲存機制

- 鏈上 vs 鏈下儲存權衡
  - 成本與可行性分析
  - 永久性保證
- IPFS 分散式儲存
  - 內容定址機制
  - CID 內容識別碼
  - 網關依賴問題
- Arweave 永久儲存
  - 一次付費永久儲存
  - 區塊編織技術
- 中心化儲存風險
- 元數據不可變性問題
- Frozen Metadata 凍結元數據

### NFT 鑄造與交易

- Minting 鑄造流程
- Gas War 搶購現象
- 荷蘭拍賣與公平發行
- Lazy Minting 延遲鑄造
- NFT 市場機制
  - 固定價格銷售
  - 拍賣機制
  - 版稅 Royalty 機制
- 流動性問題與碎片化

### NFT 進階應用

- 動態 NFT Dynamic NFT
- Soulbound Token 靈魂綁定代幣
- NFT Fi 金融化應用
  - NFT 抵押借貸
  - NFT 碎片化
  - NFT 租賃
- 元宇宙與虛擬世界應用
