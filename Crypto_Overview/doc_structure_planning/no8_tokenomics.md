# 第八階段：代幣經濟學

## 階段目標

探討代幣經濟模型設計，理解如何透過經濟激勵機制設計可持續的加密貨幣生態系統。

---

## 核心主題

### 代幣經濟學基礎

- Tokenomics 定義與重要性
- 代幣分類
  - Utility Token 功能型代幣
  - Security Token 證券型代幣
  - Governance Token 治理代幣
- 代幣供應模型
  - 固定供應 vs 動態供應
  - 總供應量與流通供應量
- 價值捕獲機制
- 需求驅動因素分析

### 代幣分配機制

- 初始分配策略
  - 團隊分配與鎖倉
  - 投資人分配
  - 社群分配
  - 生態系統儲備
- Vesting 解鎖機制
  - Cliff 懸崖期
  - 線性解鎖 vs 階段解鎖
- Airdrop 空投策略
  - 目標受眾選擇
  - 女巫攻擊防護
  - Retroactive Airdrop 追溯空投
- 募資模式
  - ICO Initial Coin Offering
  - IEO Initial Exchange Offering
  - IDO Initial DEX Offering
  - 公平發行 Fair Launch

### 通膨與通縮模型

- 固定供應模型：比特幣案例
- 通膨模型：以太坊早期案例
- 通縮模型設計
  - 代幣燃燒機制
  - 交易費用燃燒
  - EIP-1559 燃燒機制分析
- 回購機制
- 通膨率動態調整
- 經濟永續性分析
- 長期激勵設計

### 治理代幣機制

- 治理權與投票權分配
- 提案與投票流程
- 時間加權投票 veToken 模型
  - Curve veCRV 案例分析
  - 鎖倉時間與投票權重
- 委託投票機制
- Snapshot 鏈下投票工具
- 治理攻擊防護
  - Flash Loan 治理攻擊
  - 法定人數與時間鎖
- 治理激勵設計

### 激勵機制設計

- 流動性挖礦激勵
- Staking 質押獎勵
- 通膨獎勵與稀釋效應
- 激勵可持續性問題
- 吸血鬼攻擊 Vampire Attack
- 代幣經濟飛輪效應
