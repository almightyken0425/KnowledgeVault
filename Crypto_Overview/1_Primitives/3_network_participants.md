# Network Participants 網路參與者

在區塊鏈的去中心化網路中，不同的參與者扮演著不同的角色，共同維護著這本公開帳本的安全與運作。

---

## Wallets & Addresses 錢包與地址

使用者透過錢包與區塊鏈互動，但錢包裡並沒有存放任何實體的數位硬幣。

- **功能:** 錢包本質上是一個密鑰管理工具，它負責安全地儲存使用者的私鑰與公鑰。
- **地址 Address:** 這是別人轉帳給你時需要的收款帳號。在技術上，地址通常是公鑰經過雜湊運算後產生的字串。
- **餘額:** 錢包顯示的餘額，其實是程式掃描整本帳本歷史紀錄，計算出該地址所有未花費的交易輸出後的結果。

## Nodes 節點

節點是構成區塊鏈網路的電腦伺服器，它們是帳本的守護者。

- **全節點 Full Nodes:** 下載並保存了從創世區塊以來的完整區塊鏈資料。
- **驗證:** 當收到一筆新交易或新區塊時，全節點會依據程式規則進行獨立驗證。例如檢查發起人的餘額是否足夠。
- **廣播:** 驗證通過後，節點會將資訊傳遞給鄰近的節點，確保資訊在全網同步。

## Miners 礦工

礦工是特殊類型的節點，他們除了驗證交易外，還負責打包交易並產出新區塊。

- **職責:** 收集網路上待確認的交易，將它們打包進候選區塊中。
- **競爭:** 投入大量的電力與計算能力來解決工作量證明的數學難題。
- **獎勵:** 成功挖出新區塊的礦工將獲得區塊獎勵 Block Reward 與交易手續費，這是驅使他們維護網路安全的經濟誘因。
