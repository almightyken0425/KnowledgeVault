# Convolutional Layer 卷積層

這是 **電腦視覺 Computer Vision** 的靈魂核心。與傳統的全連接層 Fully Connected Layer 不同，它不是一次看整張圖片，而是像人類閱讀一樣，用一個小視窗掃描圖片。

核心概念是 **局部特徵提取 Local Feature Extraction**：
> 透過一個小的濾鏡 Filter 在圖片上滑動，尋找特定的圖案，如垂直線、水平線或圓形。

## 核心元件

### 濾鏡 Filter / Kernel
這是一個小的權重矩陣，通常是 3x3 或 5x5 的大小。每一個濾鏡負責尋找一種特定的特徵。
- **邊緣濾鏡:** 專門對圖片中的邊緣有反應。
- **顏色濾鏡:** 專門對圖片中的紅色區塊有反應。

### 滑動視窗 Sliding Window
濾鏡從圖片的左上角開始，一步一步往右滑，滑到底後再往下滑。這保留了像素之間的 **空間關係 Spatial Relationship**。

### 特徵圖 Feature Map
這是卷積運算的輸出結果。如果不使用 10 個不同的濾鏡去掃描同一張圖片，就會產生 10 張不同的特徵圖，疊在一起形成一個立體的輸出層。

## 運作參數 Hyperparameters

- **步幅 Stride:** 濾鏡每次移動幾格。
    - Stride = 1 表示一格一格細緻掃描。
    - Stride = 2 表示跳著掃描，輸出圖片會變小。
- **填補 Padding:** 在圖片外圍補一圈 0。
    - 目的是為了讓輸出圖片跟輸入圖片保持一樣大小，避免邊緣資訊流失。

## 與全連接層的比較

| 特性         | Fully Connected Layer                   | Convolutional Layer                         |
| :----------- | :-------------------------------------- | :------------------------------------------ |
| **連接方式** | **全連接** 每個輸入像素都連到每個神經元 | **局部連接** 每個神經元只看圖片的一小塊區域 |
| **參數數量** | **巨大** 處理大圖時參數會爆炸           | **極少** 同一個濾鏡在整張圖上共用權重       |
| **空間感**   | **無** 必須把圖片展平成一長串數字       | **有** 保留了圖片的長寬結構                 |

## 應用與直覺

想像你在找一張照片裡有沒有 **貓**：
- **第一層卷積:** 發現了很多 線條 和 圓弧。
- **第二層卷積:** 這些線條組成了 眼睛 和 耳朵 形狀。
- **第三層卷積:** 這些形狀組成了 貓臉。

這就是深度的意義，特徵由簡單到複雜層層堆疊。
