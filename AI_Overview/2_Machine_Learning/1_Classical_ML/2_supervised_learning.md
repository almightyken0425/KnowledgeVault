# Supervised Learning 監督式學習

這是目前產業應用最成熟、價值最高的機器學習類型。它的核心是 **有標準答案的學習 Learning with Labels**。

核心概念是 **函數逼近 Function Approximation**：
> 給定一組輸入 x 與對應的正確輸出 y，尋找一個數學函數 f，使得 f of x 盡可能接近 y。

## 學習過程

這就像是 **老師與學生** 的關係：
- **老師出題:** 給模型一張照片。
- **學生作答:** 模型猜測這是什麼。
- **老師批改:** 告訴模型正確答案。
- **學生修正:** 模型根據誤差調整內部參數。

## 兩大主要任務

根據 **輸出 Output** 的型態不同，監督式學習分為兩類：

### 回歸 Regression
預測 **連續數值 Continuous Values**。

- **目標:** 預測明天的氣溫是幾度？房價是多少錢？
- **輸出:** 25.5 度、3000 萬。
- **代表演算法:** 線性回歸 Linear Regression。
- **視覺化:** 試圖畫出一條線穿過所有數據點。

### 分類 Classification
預測 **離散類別 Discrete Categories**。

- **目標:** 這封信是垃圾郵件嗎？這張圖是貓還是狗？
- **輸出:** 是/否、貓/狗/鳥。
- **代表演算法:** 邏輯回歸 Logistic Regression、支持向量機 SVM、決策樹 Decision Tree。
- **視覺化:** 試圖畫出一條邊界線 Decision Boundary 將不同顏色的數據點分開。

## 關鍵挑戰

### 過擬合 Overfitting
模型像個死記硬背的學生，把考古題 訓練資料 的每一個細節都背下來，但在期末考 測試資料 卻考得很差。它學到了雜訊而不是規律。

### 欠擬合 Underfitting
模型太笨或太簡單，連考古題都做不好。

## 總結

| 任務     | 輸出類型    | 範例               |
| :------- | :---------- | :----------------- |
| **回歸** | 數字 多少   | 預測股價、預測銷量 |
| **分類** | 類別 是什麼 | 刷臉解鎖、手寫辨識 |
