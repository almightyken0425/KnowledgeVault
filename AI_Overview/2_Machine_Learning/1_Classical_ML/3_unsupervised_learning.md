# Unsupervised Learning 非監督式學習

如果說監督式學習是 **老師教學生**，那麼非監督式學習就是 **讓學生自己看書找規律**。

核心概念是 **發現結構 Discovering Structure**：
> 給定一組輸入 x 但 **沒有** 對應的標籤 y，演算法必須自行在數據中找出隱藏的模式或群聚。

## 為什麼需要非監督式學習？

現實世界中，大部分的數據都是 **沒有標籤 Unlabeled** 的。標註數據非常昂貴且耗時，例如請醫生標註 X 光片，或請翻譯官標註雙語語料。非監督式學習讓我們能利用海量的未標註數據。

## 兩大主要任務

### 分群 Clustering

將相似的資料點歸類在一起。

- **目標:** 這些顧客可以分成哪幾類？這篇文章屬於財經還是體育版？
- **運作:** 無需事先定義 **類別**，演算法會根據數據點之間的 **距離** 自動分組。
- **代表演算法:** K-Means K-平均演算法、DBSCAN 密度分群。
- **應用:**
    - **顧客細分:** 找出高消費潛力客群。
    - **異常偵測:** 找出信用卡盜刷行為 因為它們與正常行為的距離很遠。

### 降維 Dimensionality Reduction

在保留最重要資訊的前提下，減少資料的特徵數量。

- **目標:** 將 100 個特徵壓縮成 2 個，以便人類可以在二維平面上畫出來觀察。
- **運作:** 找出資料變異量最大的方向，將數據投影到該方向上。
- **代表演算法:** PCA 主成分分析、t-SNE。
- **應用:**
    - **資料視覺化:** 將高維度數據畫成圖表。
    - **壓縮:** 減少儲存空間與運算量。

## 與監督式學習的比較

| 特性         | Supervised 監督式              | Unsupervised 非監督式  |
| :----------- | :----------------------------- | :--------------------- |
| **輸入數據** | **有標籤** Input + Label       | **無標籤** Input only  |
| **目標**     | 預測 Predicting                | 探索 Exploring         |
| **評估指標** | **準確率 Accuracy** 答案對不對 | **聚合度** 分得好不好  |
| **難度**     | 相對簡單，有明確目標           | 相對困難，沒有標準答案 |

## 總結

非監督式學習通常是用來 **輔助** 監督式學習的。例如先用 **降維** 去除雜訊特徵，再丟進監督式模型訓練；或是先用 **分群** 自動產生標籤 偽標籤 Pseudo-Labeling，再進行微調。
