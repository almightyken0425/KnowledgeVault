# 4. MVP 範圍定義

_(本文件定義 V1 上線必須包含的最小功能集)_

## 1. 核心功能 (In Scope for MVP)

- [ ] **使用者認證:** 強制 Google 登入 (Firebase Auth)。
    
- [ ] **資料同步:** Firestore 即時同步 (實現無痛換機)。
    
- [ ] **核心記帳:**
    
    - [ ] 新增/編輯/刪除 **支出 (Expense)** 交易。
        
    - [ ] 新增/編輯/刪除 **收入 (Income)** 交易。
        
    - [ ] 新增/編輯/刪除 **同幣別** 轉帳 (Transfer)。
        
    - [ ] 新增/編輯/刪除 **跨幣別** 轉帳 (Transfer)。
        
- [ ] **多幣別功能:**
    
    - [ ] 建立/管理外幣帳戶。
        
    - [ ] 手動匯率管理介面。
        
    - [ ] 報表/總覽中的匯率換算 (依最新匯率)。
        
- [ ] **自動化:**
    
    - [ ] **定期交易 (`Schedules`)** 的建立、編輯、刪除。
        
    - [ ] App 啟動時的定期交易自動產生邏輯。
        
- [ ] **首頁 (HomeScreen):**
    
    - [ ] 預設顯示主要帳戶、當日數據。
        
    - [ ] 時間區間切換 (日/週/月/年，含左右滑動)。
        
    - [ ] 帳戶篩選器 (可選擇納入統計的帳戶)。
        
    - [ ] 視圖切換 (摘要/列表)。
        
    - [ ] **摘要視圖:** 圓餅圖、收支結餘、分類佔比摘要 (水平滾動+展開)。
        
    - [ ] **列表視圖:** 交易列表 (按日期分組 **及** 按類別分組)。
        
- [ ] **帳戶管理 (CRUD):**
    
    - [ ] 建立/查看/編輯/刪除帳戶 (免費版限制 3 個)。
        
    - [ ] 設定主要帳戶 (`IsPrimary`)。
        
- [ ] **類別管理 (CRUD):**
    
    - [ ] 建立/查看/編輯/刪除自訂類別 (免費版限制 10 個)。
        
    - [ ] 建立時必須映射到 `StandardCategoryId`。
        
    - [ ] 包含 Icon 選擇器介面。
        
- [ ] **首次啟動流程:**
    
    - [ ] 偵測新用戶。
        
    - [ ] 建立預設幣別、時區、主要帳戶、預設類別。
        
- [ ] **設定 (MVP):**
    
    - [ ] 帳戶管理入口。
        
    - [ ] 類別管理入口。
        
    - [ ] 匯率管理入口。
        
    - [ ] 偏好設定入口 (包含主要貨幣、時區、語系切換)。
        
    - [ ] 登出功能。
        
    - [ ] 清除資料功能。
        
    - [ ] 匯入資料 (`.csv` 匯入)。
        
- [ ] **其他核心功能:**
    
    - [ ] **搜尋功能 (`Note` 搜尋)**。
        
- [ ] **國際化 (i18n):**
    
    - [ ] 支援至少兩種語言 (例如 繁體中文、英文) 的介面切換。
        
- [ ] **訂閱與付費機制:**
    
    - [ ] 偵測免費版限制 (帳戶/類別)。
        
    - [ ] 提示升級付費版。
        
    - [ ] 串接 App Store / Google Play 訂閱機制。
        

## 2. 延後功能 (Out of Scope for MVP)

- [x] **匯出資料** (`.csv` / `.db`)。
    
- [x] **快速捕捉小工具 (Widgets)**。
    
- [x] (未來) **AI 分析**。
    
- [x] (未來) **進階報表:** 獨立的「報表」頁面 (長條圖、趨勢圖等)。
