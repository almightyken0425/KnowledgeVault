##### 使用流程

**玩家觀點**

| 狀態 | 通知管道 |
|------|----------|
| 未登入未註冊 — 各種優惠 | 系統外、置中廣告、首頁 Banner |
| 未登入未註冊 — 主推遊戲 | 熱門遊戲 |
| 註冊 — 首存優惠 | 註冊 PopUp |
| 註冊 — 優惠領取 | 消息中心、一般優惠、左側展開 |
| 註冊 — 歡迎光臨 | 公告中心、消息中心 |
| 登入 — 歡迎回來 | 公告中心、消息中心 |
| 登入 — 注意事項 | 公告 |
| 登入 — 公告優惠 | 公告 |
| 活躍中 — 存款相關 | 公告中心、消息中心、存款頁 |
| 活躍中 — 下注相關 | 首頁遊戲、左側展開、網頁通知 |
| 未登入已註冊 — 救援金 | 系統外、App 通知、Email 通知、SMS 通知 |
